Pero ... ¿Que es un static site generator?

  • Permiten desarrollar como si fuera un sitio dinámico.
  • Permiten hacer deploy de el sitio como si fuera uno estático.
  • El contenido se puede generar en formatos alternativos al HTML (Markdown, Texttile)

Existen en varios lenguajes

Ruby, JavaScript, Python, Go, PHP, Etc.
